# Redesign Reference

Esta pasta contém o frontend standalone gerado pelo Google AI Studio e materiais de referência visual.

## Objetivo
Servir como referência para integrar o novo design nas páginas da feature de atribuição de aulas.

## Regras
- O conteúdo desta pasta NÃO é código de produção pronto para substituir o sistema atual.
- O conteúdo deve ser tratado como referência visual e estrutural.
- O sistema atual continua sendo a base funcional real.
- O redesign deve ser modularizado e integrado com segurança.

## Estrutura
- `google-ai-studio/`: projeto standalone gerado pelo AI Studio
- `references/`: imagens e documentos de referência visual
